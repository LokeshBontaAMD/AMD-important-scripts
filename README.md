# AMD Important Scripts

A collection of useful scripts for AMD GPU development and setup.

## Folders

| Folder | Description |
|--------|-------------|
| [installation/](installation/) | TheRock nightly tarball auto-installer and verification scripts |
| [set_rocm_env/](set_rocm_env/) | ROCm/HIP environment setup script for custom TheRock installs |
| [rpp_related/](rpp_related/) | RPP build helper script (HIP/CPU/OCL backends, pre-flight checks) |
